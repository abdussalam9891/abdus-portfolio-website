# Abdus Salam — Frontend Developer Portfolio

Personal portfolio showcasing my projects, skills, and journey into frontend development.

## Live
[abdus-portfolio-website.vercel.app](https://abdus-portfolio-website.vercel.app)

## Built With
- Next.js (App Router)
- Tailwind CSS
- shadcn/ui

## Features
- Dark / light mode with localStorage persistence
- Fully responsive across all devices
- Smooth scroll navigation
- Project showcase with live links

## Run Locally
git clone https://github.com/abdus-salam/abdus-portfolio-website
cd abdus-portfolio-website
npm install
npm run dev
